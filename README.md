# CMC Materials (cmc-materials)
CMC Materials is a global supplier of consumable materials primarily to the semiconductor industry, including chemical mechanical planarization (CMP) slurries and polishing pads. Acquired by Entegris in 2022, CMC's materials portfolio is now part of the Entegris specialty chemicals and CMP solutions business.
